# Demographic Data Analyzer

This project is part of the freeCodeCamp Data Analysis with Python Certification.

## Features

This project analyzes demographic data using Pandas.

It calculates:

- Race distribution
- Average age of men
- Percentage with Bachelor's degrees
- Income analysis
- Minimum work hours
- Highest earning countries
- Most popular occupation in India for people earning >50K

## Technologies Used

- Python
- Pandas

## Files

- demographic_data_analyzer.py
- main.py
- requirements.txt
- adult.data.csv
- test_module.py

## Run Project

```bash
python main.py
```
